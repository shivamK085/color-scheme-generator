# 🎨 Color Scheme Generator

A simple web application that generates beautiful color palettes using the Color API.

Users can choose a base color, select a color scheme mode, and instantly generate a five-color palette. Clicking on any color copies its HEX code to the clipboard.

## ✨ Features

- Pick any base color using the color picker
- Generate multiple color schemes
- Supports:
  - Monochrome
  - Monochrome Dark
  - Monochrome Light
  - Analogic
  - Complement
  - Analogic Complement
  - Triad
  - Quad
- Displays HEX values for each generated color
- Click any color to copy its HEX value to the clipboard

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/shivamK085/color-scheme-generator.git
```

Go into the project folder

```bash
cd color-scheme-generator
```

Open `index.html` in your browser.
